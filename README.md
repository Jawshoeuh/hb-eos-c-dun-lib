A C library for dungeon generation functions that are compatible with C-of-Time (requires new version with submodule support).

# HOW TO USE
- ExtraSpace (Overlay36) needs to be applied.
- Add this repo as a submodule of your C-of-Time repo (`git submodule add https://github.com/jawshoeuh/hb-eos-c-dun-lib modules/hb-eos-c-dun-lib`)
- Include the library in the corresponding C file with '#include "hb-eos-c-dun-lib.h".
- (Optional) Review the functions provided by the library in 'hb-dun-lib.yml' or 'hb-eos-c-dun-lib.h'.
